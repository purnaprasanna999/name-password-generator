# Name-Based Password Generator 🔐

A simple Python program that generates a password based on your name.

## How it works
- User enters their name
- Name is converted to lowercase
- Random numbers and symbols are added after the name
- Final password is displayed

## Example Output
===== Name-Based Password Generator =====
Enter your name: Rahul
Enter total password length: 10

Hello, rahul!
Your Password: rahul48@#k
Keep it safe!

## Features
- Name-based personalized password
- Input validation (letters only)
- Auto-corrects length if too short
- Uses Python built-in modules only

## How to Run
1. Install Python 3 from python.org
2. Download the file name_password_generator.py
3. Open terminal or command prompt
4. Run: python name_password_generator.py

## Tech Used
- Python 3
- random module
- string module
